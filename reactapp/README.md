## Zadanie 5 - React

Należy stworzyć komponenty odpowiadające funkcjonalności użytkownika, 
tzn. nie musi być 1:1 do kontrolerów w Play. Należy stworzyć minimum 
8 komponentów, a ewentualne zapytania do serwera należy wydzielić 
do zewnętrznej  klasy i/lub serwisu.

## Zdanie 6. React hooks

Należy wykorzystać React hooks, aby udostępnić dane np. produkty pomiędzy komponentami. 
Należy wykorzystać je tam, gdzie dane z serwera sa wykorzystywane. Alternatywa: mobx, redux.
